# Contributing

This project is **source-available**. External contributions (issues, pull requests, discussions) are not accepted.

The source is published under Apache 2.0 so that users of the Vion IoT platform can read, audit, and depend on it — not as an invitation to collaborate.

For questions or problem reports, contact `info@vion-iot.com`.

For security vulnerabilities, see [SECURITY.md](SECURITY.md).
